# Thief Runner Game

A mobile-first endless runner game built with React, Vite, and Tailwind CSS.

## Setup

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run the Game:**
    ```bash
    npm run dev
    ```

## Custom Assets

To use your own images and music, place the following files in the `public` folder:

-   `thief.png` - The player character image.
-   `officer.png` - The chasing officer image.
-   `bg-music.mp3` - Background music file.

If these files are missing, the game will use colored shapes (Blue for Thief, Red for Officers).

## Controls

-   **Desktop:** Spacebar or Arrow Up to Jump.
-   **Mobile:** Tap anywhere to Jump.
